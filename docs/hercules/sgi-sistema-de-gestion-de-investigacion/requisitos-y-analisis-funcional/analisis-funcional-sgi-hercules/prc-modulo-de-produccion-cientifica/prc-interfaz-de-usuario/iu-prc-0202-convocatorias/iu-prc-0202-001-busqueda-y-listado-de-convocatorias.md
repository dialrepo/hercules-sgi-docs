# Hércules : IU\-PRC\-0202\-001 Búsqueda y listado de convocatorias



## Formulario de Búsqueda y listado de convocatorias

Pantalla que muestra un formulario, que permite aplicar filtros sobre el listado de todas las convocatorias de baremación de grupos dadas de alta en el SGI. Existirá una por año.

Actuará como filtro sobre el campo "activo" de la tabla "convocatoria baremación". |

| Modificar | Icono de acción | Acción modificar. Solo estará disponible para los elementos activos (campo "activo" a "true") y si el campo "fecha inicio ejecución" es null. |

| Ver | Icono de acción | Acción consulta. Solo estará disponible para los elementos activos (campo "activo" a "true"). |
| Clonar | Icono de acción | Acción clonar. Solo estará disponible para los elementos activos (campo "activo" a "true"). |
| Lanzar proceso de baremación | Icono de acción | Acción de lanzar el proceso de baremación. Solo estará disponible si se cumplen las siguientes condiciones:* campo "activo" a "true" * campo "anio" el año actual. * campo "fecha inicio ejecución" es null (no existe el proceso ya lanzado) o "fecha inicio ejecución" y "fecha fin ejecución" son distintas de null (se ha lanzado y finalizado el proceso).  En otro caso no se dejará lanzar el proceso y no se mostrará el icono de Lanzar proceso de baremación.Mostrar dialogo de confirmación antes de lanzar el proceso. |

## Formulario Crear/Modificar/consultar convocatoria de baremación \- Moduladores y rangos

Formulario de creación/modificación/consulta de una convocatoria de baremación, apartado de Moduladores y rangos. El usuario ACT\-PRC\-003\-Gestor podrá crear o realizar cambios.

Si el usuario es un ACT\-PRC\-004\-Visor se mostrarán todos los campos en modo consulta y no podrá realizar modificaciones. El botón Guardar estará deshabilitado.

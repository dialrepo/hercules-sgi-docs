# Hércules : IU\-PRC\-0202\-003 \- Crear\-Modificar\-Consultar convocatoria de baremación \- Baremos y puntuaciones



## Formulario Crear/Modificar/consultar convocatoria de baremación \- Baremos y puntuaciones

Formulario de creación/modificación/consulta de una convocatoria de baremación, apartado de Baremos y puntuaciones. El usuario ACT\-PRC\-003\-Gestor podrá crear o realizar cambios.

Si el usuario es un ACT\-PRC\-004\-Visor se mostrarán todos los campos en modo consulta y no podrá realizar modificaciones. El botón Guardar estará deshabilitado.

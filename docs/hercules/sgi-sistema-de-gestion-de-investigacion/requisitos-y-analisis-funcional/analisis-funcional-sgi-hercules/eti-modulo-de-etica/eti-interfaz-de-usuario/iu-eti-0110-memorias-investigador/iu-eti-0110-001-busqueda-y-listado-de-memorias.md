# Hércules : IU\-ETI\-0110\-001 Búsqueda y listado de memorias



## Formulario de Búsqueda y listado de memorias

Pantalla que muestra un formulario, que permite aplicar filtros sobre el listado de memorias de peticiones de valuación que tiene creadas un investigador o solicitante o bien en aquellas en las que el usuario es el responsable de la memoria. Son las asignadas al usuario que esta logeado en el sistema.

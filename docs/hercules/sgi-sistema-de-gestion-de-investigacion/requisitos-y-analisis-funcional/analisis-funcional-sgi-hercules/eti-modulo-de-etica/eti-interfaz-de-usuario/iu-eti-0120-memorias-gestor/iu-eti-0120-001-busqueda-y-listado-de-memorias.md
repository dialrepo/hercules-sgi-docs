# Hércules : IU\-ETI\-0120\-001 Búsqueda y listado de memorias



## Formulario de Búsqueda y listado de memorias

Pantalla que muestra un formulario, que permite aplicar filtros sobre el listado de memorias de peticiones de valuación dadas de alta en el sistema. El gestor puede visualizar todas las memorias.

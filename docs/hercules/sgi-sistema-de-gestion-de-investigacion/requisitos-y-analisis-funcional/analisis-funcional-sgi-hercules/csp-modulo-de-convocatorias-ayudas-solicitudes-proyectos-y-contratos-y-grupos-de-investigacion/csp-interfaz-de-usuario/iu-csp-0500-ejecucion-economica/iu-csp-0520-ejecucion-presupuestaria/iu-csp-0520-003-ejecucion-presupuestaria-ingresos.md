# Hércules : IU\-CSP\-0520\-003 \- Ejecución presupuestaria \- Ingresos



## Formulario Ejecución presupuestaria \- Ingresos

Formulario que mostrará el balance de las aplicaciones presupuestarias de ingreso. Se permitirá la visualización de las mismas para una o varias anualidades del proyecto. Las aplicaciones presupuestarias mostradas serán las incluidas en el presupuesto del proyecto. Las columnas de detalle mostradas para cada aplicación presupuestaria serán las devueltas por el sistema de gestión económico.

El listado podrá ser exportado a .csv.

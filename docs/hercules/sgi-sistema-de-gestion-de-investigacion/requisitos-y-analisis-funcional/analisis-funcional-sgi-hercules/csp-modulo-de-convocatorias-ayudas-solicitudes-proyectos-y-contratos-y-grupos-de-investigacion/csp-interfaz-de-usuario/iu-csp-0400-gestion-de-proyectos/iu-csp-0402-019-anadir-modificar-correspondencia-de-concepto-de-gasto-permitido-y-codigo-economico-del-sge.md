# Hércules : IU\-CSP\-0402\-019 \- Añadir\-modificar correspondencia de concepto de gasto permitido y código económico del SGE



## Formulario Añadir\-modificar correspondencia de concepto de gasto permitido y código económico del SGE

Formulario que permite añadir correspondencia de un concepto de gasto permitido del proyecto y el código económico del SGE.



| **Código económico (cuando el registro no está vinculado a un registro de la convocatoria relacionada: el campo "convocatoria concepto gasto código económico ID" no está informado)****Código económico (cuando el registro está vinculado a un registro de la convocatoria relacionada: el campo "convocatoria concepto gasto código económico ID" está informado)**En el caso de que el registro no exista en el Proyecto y si en la Convocatoria, todos los datos del proyecto estarán en modo no editable hasta que se pulse el botón de "Aplicar al proyecto" que se inicializarán los datos del proyecto con los valores de la convocatoria y se pondrán todos los campos editables por si desea modificarlos excepto el campo "Código económico SGE" que no se podrá modificar.Cuando el registro existe en el proyecto, todos los campos serán editables, excepto el campo "Código económico SGE" que no se podrá modificar. | | |

# Hércules : IU\-CSP\-0406\-025 \- Modificar proyecto \- Configuración económica \- Presupuesto



## Formulario Modificar proyecto \- Configuración económica \- Presupuesto

Formulario que permite realizar el desglose de presupuesto del proyecto, indicando si se desglosa o no en anualidades y permitiendo añadir los gastos e ingresos por anualidad. Aunque el proyecto provenga de un solicitud de convocatoria no se copiará el desglose indicado en la solicitud debido a que será necesario especificar en el proyecto el importe del presupuesto (que no tiene por qué coincidir con el previsto en la solicitud) y el finalmente concedido así como las partidas/aplicaciones presupuestarias.

Además, este botón no se mostrará cuando ya exista la anualidad genérica ( campo "anualidades" de la tabla "proyecto" a "no" y existe un registro en la tabla "proyecto anualidad").  Si el campo aún no ha sido informado (campo "anualidades" de la tabla "proyecto" no está informado) el botón "Añadir anualidad"/"Añadir anualidad genérica" se mostrará deshabilitado.Si ya hay alguna anualidad creada (existen registros en la tabla "proyecto anualidad"), no se permitirá la modificación del valor de este campo "¿Desglosar en anualidades?", mostrándose deshabilitado. |

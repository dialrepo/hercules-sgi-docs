# Hércules : IU\-CSP\-0401\-001 \- Crear proyecto \- Datos generales \- Ficha general



## Formulario Crear proyectos \- Datos generales \- Ficha general

Formulario de creación de un proyecto, apartado de Datos generales \- Ficha general. En caso de que el proyecto provenga de una convocatoria/solicitud registrada en el SGI, se recuperará información registrada previamente en la convocatoria/solicitud.

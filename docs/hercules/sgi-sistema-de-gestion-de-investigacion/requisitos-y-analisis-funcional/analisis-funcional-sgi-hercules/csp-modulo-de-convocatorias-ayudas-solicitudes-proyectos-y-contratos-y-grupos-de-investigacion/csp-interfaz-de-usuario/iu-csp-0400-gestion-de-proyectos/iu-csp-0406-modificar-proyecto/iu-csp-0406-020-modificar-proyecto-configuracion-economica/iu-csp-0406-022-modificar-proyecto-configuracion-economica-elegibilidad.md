# Hércules : IU\-CSP\-0406\-022 \- Modificar proyecto \- Configuración económica \- Elegibilidad



## Formulario Modificar proyecto \- Configuración económica \- Elegibilidad

Formulario que permite realizar la configuración de los conceptos de gasto que van a estar habilitados en el proyecto: gastos elegibles o permitidos, identificación de concepto de gasto que representa a los costes indirectos y conceptos de gasto no elegibles o no permitidos. Si el proyecto ha sido creado a partir de la concesión de una solicitud o está vinculado a una convocatoria esta información podrá ser recuperada de la convocatoria.

# Hércules : IU\-CSP\-0302\-006 \- Añadir\-modificar miembro a equipo del socio



## Formulario Añadir\-modificar miembro a equipo del socio

Formulario que permitirá añadir o modificar, durante el proceso de creación o modificación de una solicitud de proyecto, un miembro al equipo de proyecto de un socio.



|  | | | |
| --- | --- | --- | --- |
| Nombre | | Tipo | Características / Notas |
| Rol de participación | | SelectorTexto corto | Rol con el que participará el miembro del equipo del socio colaborador. Se deberá indicar obligatoriamente un rol para cada uno de los miembros del equipo del socio colaborador. Los roles disponibles serán los configurados en la tabla "Rol proyecto"Cuando este formulario "Miembro equipo socio" se abre desde la opción de modificación de un miembro que ya figure en el listado del equipo, el combo se mostrará cargado con el rol asociado en el listado y en caso de modificar su valor, se deberá de realizar una invocación al servicio asociado al buscador de personas, de forma que se compruebe que el miembro ("persona ref") pertenezca a uno de los colectivos asociados al rol seleccionado (tabla "rol proyecto colectivo"). Si la búsqueda del miembro sobre los colectivos del rol no devuelve valor, será indicativo de que el miembro no puede tener el nuevo rol, por lo que se limpiará el valor mostrado en el campo "Miembro" obligando a realizar una nueva búsqueda de persona. |

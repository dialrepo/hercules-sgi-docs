# Hércules : IU\-CSP\-0306\-004 \- Modificar solicitud \- Requisitos convocatoria



## Formulario Modificar solicitud \- Requisitos convocatoria

Formulario de modificación de solicitud de RRHH, apartado Requisitos convocatoria, para todos los perfiles:  ACT\-CSP\-001\-Investigador, ACT\-CSP\-002\-InvestigadorExterno (usuario externo) o para los perfiles propios de la unidad de gestión (ACT\-CSP\-003\-Gestor, ACT\-CSP\-004\-Administrador)  


El ACT\-CSP\-001\-Investigador y ACT\-CSP\-002\-InveestigadorExterno únicamente podrá modificar la solicitud mientras ésta se encuentre en estado "**Borrador**" y "**Rechazada**"

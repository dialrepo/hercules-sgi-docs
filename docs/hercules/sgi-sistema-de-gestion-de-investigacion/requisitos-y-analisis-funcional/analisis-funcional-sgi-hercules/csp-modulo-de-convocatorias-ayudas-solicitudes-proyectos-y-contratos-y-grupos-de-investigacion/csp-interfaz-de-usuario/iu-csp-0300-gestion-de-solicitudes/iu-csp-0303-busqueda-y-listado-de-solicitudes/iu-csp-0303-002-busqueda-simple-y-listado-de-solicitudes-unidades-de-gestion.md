# Hércules : IU\-CSP\-0303\-002\- Búsqueda simple y listado de solicitudes \- Unidades de gestión



## Formulario Búsqueda y listado de solicitudes \- Unidades de gestión

Formulario que muestra los filtros que se pueden realizar sobre las solicitudes y el listado de solicitudes que cumplen los criterios de búsqueda. Automáticamente se aplicará el filtro de la Unidad de Gestión, de forma que solo se obtendrán las solicitudes asociadas a a las unidades de gestión a las que esté vinculado el usuario.
